Testes

Obs. Para garantir os testes é recomendado apagar o "arquvivo.txt" antes de inicias, pois ele salva 
as operacoes efetuadas no banco

Teste Ok e NOK
Descricao
 - Vao ser inseridos valores para testar e as repostas do servidor
 - O OK para testar sucesso e NOK falha

Como executar
 - Os servidor tem que estar online
 - Entre na pasta do projeto pelo terminal 
 - Execute o script "test-oknok.sh"

Teste OPERACOES
Descricao
 - Vai ser feitas todas as operacoes em 15 chaves (2 a 16)
 - Primeiro um DELETE e depois um CREATE
 - Por fim um UPDATE
 - Vao ser inseridos 15 chaves
 - No final o banco deve ter os valores de A a O

Como executar
 - Servidor online
 - Entre na pasta do projeto pelo terminal
 - Execute o script "test-operacoes.sh"
